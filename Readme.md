# TodoList

> 기술스택


* backend
    * node.js
    * express(memory DB)

* frontend
    * react.js
    * axios
    * prop-types
    * redux
    * redux-saga
    * styled-components


> 데모

https://todolist-sigma.now.sh/


> 개발환경 세팅 및 실행

* node.js v12.16.1 install

https://nodejs.org/dist/v12.16.1/node-v12.16.1-x64.msi


* git clone

```shell script
git clone https://github.com/JunH-K/todolist.git
```

* /todolist 경로에서 npm install

```shell script
npm install
```

* 실행
```shell script
npm run dev
```
